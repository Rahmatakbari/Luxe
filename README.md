## ✨ ویژگی‌ها

### 🎨 طراحی
- ✅ رابط کاربری مدرن با گرادیانت‌های زیبا
- ✅ **حالت شب/روز** (Dark Mode) با ذخیره‌سازی

- ✅ صفحه جزئیات محصول با محصولات مشابه

- ✅ **لیست علاقه‌مندی** (Wishlist)

### 🔐 پنل ادمین
- ✅ داشبورد با آمار زنده
- ✅ مدیریت محصولات، سفارشات، مشتریان

- ✅ تنظیمات فروشگاه

### 💾 ذخیره‌سازی

## 📁 ساختار پروژه

```
shop/
├── index.html          # صفحه اصلی
├── shop.html           # فروشگاه با فیلتر
├── product.html        # جزئیات محصول
├── cart.html           # سبد خرید
├── checkout.html       # صفحه پرداخت
├── login.html          # ورود / ثبت‌نام
├── admin.html          # پنل مدیریت
├── css/
│   └── style.css       # تمام استایل‌ها + Design System
├── js/
│   ├── data.js         # داده محصولات + توابع کمکی
│   └── app.js          # منطق سبد، تم، Wishlist، Toast
└── README.md
```

## 🚀 راه‌اندازی

فقط فایل `index.html` رو در مرورگر باز کنید — همین!

نیازی به سرور، نصب پکیج یا بیلد نیست.

### توصیه (اختیاری):
برای بهترین تجربه، از یک سرور محلی استفاده کنید:
```bash
# با Python
python -m http.server 8000

# با Node.js
npx serve
```
