# 📝 راهنمای اجرای Luxe در Quick Edit (اندروید)

راهنمای گام‌به‌گام برای اجرا و تست پروژه Luxe در گوشی اندروید با Quick Edit

---

## 📥 مرحله ۱: آماده‌سازی فایل‌ها

### گزینه A: از GitHub
1. برید به GitHub Repository تون
2. روی دکمه سبز **"Code"** بزنید
3. **"Download ZIP"** رو انتخاب کنید
4. فایل ZIP رو در گوشی extract کنید

### گزینه B: انتقال از کامپیوتر
- با USB، بلوتوث یا Google Drive منتقل کنید
- در پوشه **Internal Storage / Download** بذارید

---

## 📝 مرحله ۲: باز کردن در Quick Edit

1. **Quick Edit** رو باز کنید
2. سه خط بالا سمت راست (☰) → **Open File**
3. برید به پوشه **luxe-final**
4. روی **`index.html`** بزنید

---

## 🚀 مرحله ۳: اجرا و مشاهده

### روش ۱: Preview داخلی (سریع‌ترین)
1. در Quick Edit، بالای صفحه **آیکون Preview** (چشم 👁 یا مثلث ▶) رو بزنید
2. اگه نداشت: **Menu ⋮** → **View HTML** یا **Preview**

### روش ۲: باز کردن در مرورگر (توصیه میشه)
1. **Menu ⋮** → **Open with...**
2. **Chrome** یا **مرورگر پیش‌فرض** رو انتخاب کنید
3. سایت با تمام امکاناتش باز میشه ✨

### روش ۳: مدیر فایل
1. از مدیر فایل گوشی، به پوشه luxe-final برید
2. روی `index.html` بزنید
3. **Chrome** رو انتخاب کنید

---

## ✏️ مرحله ۴: ویرایش کد

### فایل‌های مهم برای ویرایش:

| فایل | چه چیزی داخلش هست |
|------|-------------------|
| `index.html` | صفحه اصلی + محصولات نمایش داده شده |
| `shop.html` | لیست کامل محصولات + فیلترها |
| `product.html` | جزئیات محصول + نظرات |
| `cart.html` | سبد خرید |
| `checkout.html` | پرداخت |
| `admin.html` | پنل مدیریت |

### 🔍 برای تغییر محصولات:

در **همه فایل‌ها** دنبال این خط بگردید:
```javascript
const PRODUCTS = [
```

بعد میتونید محصولات رو تغییر بدید:
```javascript
{id:1, img:'images/iphone.jpg', title:'اسم محصول من', price:5000, ...}
```

**نکته:** برای هماهنگی، در **همه فایل‌ها** یک تغییر رو بدید.

### 🎨 برای تغییر رنگ‌ها:

در بالای هر فایل دنبال این بگردید:
```css
:root {
  --brand: #2563eb;    /* رنگ آبی اصلی */
  --brand-2: #0ea5e9;  /* رنگ آبی روشن */
```

مثلاً برای سبز کردن:
```css
--brand: #10b981;
--brand-2: #34d399;
```

## ⚠️ محدودیت‌های اجرا در Local (بدون سرور)

این چیزها **در local کار نمیکنن** (طبیعیه):

| قابلیت | دلیل |
|--------|------|
| ❌ Service Worker (PWA) | فقط با HTTPS |
| ❌ نصب اپلیکیشن (Install PWA) | نیاز به HTTPS |
| ❌ حالت آفلاین | نیاز به Service Worker |
| ❌ عکس‌های Unsplash (کوله + کتاب) | نیاز به اینترنت |
| ⚠️ LocalStorage | معمولاً کار میکنه اما در بعضی مرورگرها محدود |

### ✅ این‌ها همه خوب کار میکنن:
- 🎨 طراحی و رنگ‌ها
- 🌙 Dark Mode
- 🔍 جستجو
- 🛒 سبد خرید (بدون ذخیره دائم)
- ⭐ نظرات
- 💬 چت پشتیبانی
- ⇄ مقایسه محصولات
- 🕒 محصولات اخیر
- 🎁 همه CSS و انیمیشن‌ها

## 🐛 مشکلات رایج و راه حل

### ❌ "صفحه سفید" یا "فایل باز نمیشه"
✅ **راه حل:**
- مطمئن بشید فایل با پسوند `.html` هست
- در Chrome باز کنید (نه Preview داخلی)

### ❌ "عکس‌ها نمایش داده نمیشن"
✅ **راه حل:**
- مطمئن بشید پوشه `images/` در کنار `index.html` هست
- ساختار فایل‌ها رو حفظ کنید

### ❌ "متن‌ها به هم ریخته"
✅ **راه حل:**
- Chrome رو آپدیت کنید
- بررسی کنید encoding فایل UTF-8 باشه

### ❌ "توست 'پیام' نمایش داده میشه"
✅ **راه حل:**
- Cache مرورگر رو پاک کنید (Chrome → تنظیمات → پاک کردن داده‌ها)

### ❌ "کلیک روی محصول کار نمیکنه"
✅ **راه حل:**
- بررسی کنید JavaScript فعال باشه
- در Chrome: تنظیمات → Site Settings → JavaScript → **Allowed**

## 💡 نکات پیشرفته

### 🎯 برای دیدن Console (خطاها):

اگه در Chrome موبایل هستید:
1. آدرس‌بار: **chrome://inspect**
2. یا از کامپیوتر با USB Debugging وصل کنید
3. Chrome DevTools باز کنید → Console

### 🔧 برای ذخیره تغییرات:
- **Ctrl+S** (اگه کیبورد دارید) یا آیکون 💾 در Quick Edit
- بعد از ذخیره، صفحه رو Refresh کنید تا تغییرات ببینید

### 📱 برای تست ریسپانسیو:
- در Chrome: **Menu ⋮** → **Desktop site** رو تیک بزنید
- ببینید در حالت دسکتاپ چطور نمایش داده میشه
